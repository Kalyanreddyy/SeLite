Summary

It's needed for SelBlocksGlobal. It works in Selenium IDE.

Detailed description

Nonvisual extension. It defines class TestCaseDebugContext, ported from Selenium IDE. A trivial extensions plugin for Selenium-IDE, to make SelBlocksGlobal work until Se IDE includes a patch requested at issue 5495. You'll need SelBlocksGlobal and Selenium IDE (tested with version 1.10.0 and 2.0.0).
